# Skills de knowledge

Skills que leem/mantêm conhecimento persistente — base compartilhada do time (via template → skill custom) e memória do agente sobre o desenvolvedor.

| Skill | O que faz |
|-------|-----------|
| [`knowledge-base`](knowledge-base/pt-BR/SKILL.md) | Template/bootstrap para criar uma skill **custom** de Knowledge Base compartilhada (com `write-a-skill`); distingue de `agent-memory` / `memory` |
| [`agent-memory`](agent-memory/pt-BR/SKILL.md) | Manter a memória de longo prazo do agente sobre um dev em um repositório de conhecimento — conduta, feedback, projetos, referências, decisões e sessões |
